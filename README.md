# wedding

Свадебное приглашение с персональными ссылками.

## Как использовать персональные ссылки

- Основная страница приглашения: `index.html`
- Имя гостя передаётся в URL параметром `guest`, например:
  - `index.html?guest=Анна`
- Отдельная страница для быстрого шаринга: `guests.html`
  - Отредактируйте массив `GUESTS` внизу файла и добавьте список имён.
  - Для каждого гостя доступны кнопки: поделиться, копировать ссылку, открыть Telegram/WhatsApp/Viber.
